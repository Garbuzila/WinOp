### 01.Убрать что-то
### ![2022-1-8 0-28-29](https://user-images.githubusercontent.com/85686554/148597120-6770c6dc-d74a-4e12-8938-0dac9cf98ba2.png)  
**Скрыть иконку Безопасное извлечение устройства** - Убирает иконку безопасного извлечения устройства.  
Сама иконка выглядит так:
### ![00333332022-1-8 0-53-19](https://user-images.githubusercontent.com/85686554/148603865-8f516c43-776d-4235-a5ae-1b565ec7a212.png)

**Убрать предложение магазина в меню - Открыть с помощью** - Когда кликаешь на меню Открыть с помощью, то вместе со списком используемых программ еще стоит пункт Магазин Microsoft. Твик убирает пункт.

**Другой способ обработки интернет трафика** - Альтернативный способ передачи пакетов и их контроля. Для меня скорость не поменялась. Я использую обычный режим.  

**Обработка шрифтов Win 10, как в Win 8.1 (Wn 10)** - Знакомая многим владельцам ноутбуков проблема. Шрифты исправляются не везде.

### 03.Сброс чего-то
### ![2022-1-8 1-57-9](https://user-images.githubusercontent.com/85686554/148607040-a2abe2c3-50d5-4f47-a375-213197194fbe.png)

### Контекстное меню<a name="Контекстное_меню"></a>
### ![2022-1-8 1-57-52](https://user-images.githubusercontent.com/85686554/148610699-012162f0-a2a9-409b-918f-a14b49390b8a.png)
**01.Очистить контекстное меню** - Очистить контекстное меню от мусорных пунктов, таких как "Отправить" или "Сетевой диск"  
Будте внимательны - это отключает возможность поменять иконку папки. Чтобы это сделать, придется вернуть изменения.  

**02.Рабочий стол в контекстном меню** - Добавляет пункт в контекстное меню для открытия папки Рабочего стола.  
Бывает полезно, когда сложно что-то найти на Раб. столе или если не хочется постоянно сворачивать окна до Раб.стола.  

**03.Корзина в контекстном меню Рабочего стола** - Добавляет функцию очистки Корзины Windows прямо из контекстного меню Раб.стола.  
Теперь не нужно тянуться к ярлыку Корзины - Очень удобно.  

**04.Список файлов в папке - Пункт Контекстного меню** - Добавляет функцию вывести названия всех файлов в папке в отдельный текстовый документ.  
Есть нюанс: не копирует названия папок - только файлов. Полезно, если хочешь вставить названия файлов из папки куда-то.  

**05.Получить полный доступ к файлу - Пункт Контекстного меню** - Некоторые файлы на Системном диске защищены так, что обычный пользователь ничего не может с ними сделать.  
Этот файл открывает права доступа к файлу.

### Новый план Электропитания<a name="Новый_план_Электропитания"></a>
### ![2022-1-8 1-58-9](https://user-images.githubusercontent.com/85686554/148616141-a5b360c4-68dd-444c-a239-373e1c404faf.png)
План Электропитания ***"Экстра производительность"*** - План, который мощнее любого базового Электропитания в Windows.  
Процессор отдает больше ресурсов активным приложениям. Пасивным процессам отдается низкий приоритет.  
Для ноутбуков, работающим от батареи, а не от сети использовать **не советую**.  
Короче - попробуйте. Если что - то его всегда можно полностью удалить.
